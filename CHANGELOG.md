# Changelog

All notable changes to the Strand Protocol project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-02-26

### Added

- Initial implementation of all 7 protocol modules
- StrandLink L1 frame protocol (Zig)
- StrandRoute L2 semantic routing (C + P4)
- StrandStream L3 hybrid transport (Rust)
- StrandTrust L4 model identity and crypto (Rust)
- StrandAPI L5 AI application protocol (Go)
- StrandCtl CLI tool (Go)
- Strand Cloud control plane (Go)
- Monorepo build system with top-level Makefile
- Cross-module integration test framework
- Protocol buffer schema definitions
- CI/CD pipeline configuration
- Developer documentation and contribution guidelines
