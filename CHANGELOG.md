# Changelog

All notable changes to the Nexus Protocol project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-02-26

### Added

- Initial implementation of all 7 protocol modules
- NexLink L1 frame protocol (Zig)
- NexRoute L2 semantic routing (C + P4)
- NexStream L3 hybrid transport (Rust)
- NexTrust L4 model identity and crypto (Rust)
- NexAPI L5 AI application protocol (Go)
- NexCtl CLI tool (Go)
- Nexus Cloud control plane (Go)
- Monorepo build system with top-level Makefile
- Cross-module integration test framework
- Protocol buffer schema definitions
- CI/CD pipeline configuration
- Developer documentation and contribution guidelines
